# **App Name**: Persona Site

## Core Features:

- Home Section: Displays your name, profession, and a brief introductory message. This section should be the first thing visitors see.
- About Me Section: Presents a detailed overview of your background, skills, and experiences.
- Projects Showcase: Displays a list of key projects with descriptions and links to live sites or repositories.
- Skills Section: Highlights your key technical and soft skills.
- Contact Form: Allows visitors to send you messages directly through the website.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and sophistication.
- Background color: Light gray (#F0F2F5), nearly white, to keep the design uncluttered.
- Accent color: Soft coral (#FF7043) to add a touch of warmth and personality.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern aesthetic.
- Use simple, outline-style icons to represent different sections and skills.
- Employ a clean, single-page layout with clear sections for easy navigation.
- Use subtle scroll animations to reveal content and create a dynamic user experience.
